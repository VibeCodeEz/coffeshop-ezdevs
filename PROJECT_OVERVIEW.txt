PROJECT PURPOSE:
----------------
A comprehensive full-stack coffee shop e-commerce website that serves as a 
complete digital solution for coffee retail businesses. This project 
demonstrates modern web development practices while providing real-world 
functionality for coffee shops to manage their online presence and sales.

MAIN OBJECTIVES:
---------------
1. Create a professional coffee shop website with e-commerce capabilities
2. Implement user authentication and role-based access control
3. Provide comprehensive product management and ordering system
4. Demonstrate modern React/TypeScript development practices
5. Integrate with Supabase for backend services and real-time data
6. Deliver a responsive, mobile-friendly user experience
